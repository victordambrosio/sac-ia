❓ Dúvidas frequentes

Precisa saber programar?
Não. É tudo feito por formulários visuais no portal do ML e aqui.
Tem custo?
A API do ML é gratuita para vendedores. O custo da IA (Claude) é de aproximadamente R$ 0,02 por resposta gerada.
O token expira?
O access token expira em 6h, mas com o scope offline_access o sistema renova automaticamente. Você não precisa autorizar novamente.
E se o Redirect URI der erro?
O valor exibido no Passo 1 já é gerado a partir da URL exata onde você está acessando este app. Use exatamente ele no campo do ML.
